1) Pengertian HTML

HTML atau Hyper Text Markup Language merupakan sebuah bahasa markup atau dapat diartikan sebagai bahasa komputer yang terstruktur dikembangkan untuk membuat halaman website yang dapat diakses atau ditampilkan menggunakan web browser. HTML sendiri secara resmi lahir pada tahun 1989 oleh Tim Berners Lee dan dikembangkan oleh World Wide Web Consortium (W3C), yang kemudian pada tahn 2004 dibentuklah Web Hypertext Application Tecnology Working Group (WHATG) yang hingga kini bertanggungjawab akan perkembangan bahasa HTML ini.
